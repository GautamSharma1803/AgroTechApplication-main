# Debug "Failed to Fetch" Error

## Check Browser Console:
1. Press F12 (or Right-click → Inspect)
2. Click "Console" tab
3. Try signing up again
4. Look for red error messages
5. Share the exact error message you see

## Common Error Messages:

### Error 1: "CORS policy"
- Backend CORS issue
- Solution: Check server CORS settings

### Error 2: "Failed to fetch" or "Network request failed"
- Can't reach backend
- Solution: Check backend URL

### Error 3: "404 Not Found"
- Wrong endpoint
- Solution: Check API route

### Error 4: "500 Internal Server Error"
- Backend error
- Solution: Check Supabase logs
